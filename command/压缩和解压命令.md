# Linux 下常用的一些压缩解压命令

## 1. 压缩或打包命令

1. 格式： **.tar**

    ```
    tar –cvf filename.tar FileOrDir
    ```

2. 格式：**.tar.gz**

    ```
    tar –zcvf filename.tar.gz FileOrDir
    ```

4. 格式：**.zip**

    ```
    zip filename.zip FileOrDir
    ```

5. 格式：**.rar**

    ```
    rar a filename.rar FileOrDir
    ```

## 2. 解压缩或解包命令

1. 格式： **.tar**

    ```
    tar –xvf filename.tar
    ```

2. 格式：**.tar.gz**

    ```
    tar -zxvf filename.tar.gz
    ```

3. 格式：**.zip**

    ```
    unzip filename.zip 
    ```

4. 格式：**.rar**

    ```
    unrar e filename.rar
    ```
